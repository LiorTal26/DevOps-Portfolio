## Animations Included

1. **Loading Screen**: Spinning loader with fade-out
2. **Typing Effect**: Animated text typing in hero section
3. **Particle System**: Animated background particles with connections
4. **Floating Icons**: Parallax floating tech icons
5. **Skill Bars**: Progress bars that animate on scroll
6. **Project Cards**: Reveal animations and hover effects
7. **Scroll Indicator**: Animated mouse and arrows
8. **Navbar**: Background changes on scroll
9. **Hover Effects**: Multiple interactive hover animations
10. **Section Titles**: Animated line reveals

## Customization

1. **Personal Information**: Update content in `index.html`
2. **Projects**: Modify project cards with your own projects
3. **Skills**: Adjust skill levels and categories
4. **Contact Links**: Update social media and email links
5. **Colors**: Modify CSS variables in `styles.css`
6. **Animations**: Adjust animation speeds and effects in CSS and JS

## Usage

1. Open `index.html` in a web browser
2. No build process required!
3. Deploy to GitHub Pages, Netlify, Vercel, or any static hosting

## Technologies
